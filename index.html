What are Semantic Elements?

Semantic elements in HTML give clear meaning to your web page for both the user (developer) and to the browser. To create a blog or forum, for example, on your page using an "Article" element (which falls under content sectioning that creates a broad outline of your page) would be an appropriate and concise directive to allow both the browser and any subsequent developer to readily understand what you have created on the page. "Article" elements are used to be independently distributable and reusable meaning you can create an ongoing blog to send to your targeted lists and allow them to find a blog history located on your site. There are sixteen main element groups to utilize on any given page all of which contain numerous start and end tag directives to help form and create your page without which there could not be communication with the web at large.
